This folder contains the data and the Python codes mentioned in the paper entitled 'Artificial Neural Network Models for Predicting the Expected Peak Displacement Of Seismic Isolation Systems Using Single Friction Pendulum Bearings'
(DOI: to be updated)
---------------------------
Folder PythonANN contains the Python codes for predicting peak displacement of isolaton systems with single bearings.
All files in this folder must be placed in the same folder.
[+] SFPB_PeakDisp.py: Contains prediction functions.
[+] examples.py: Demonstrates the usage of the prediction functions in SFPB_PeakDisp.py
***Required packages:
    [+] numpy
    [+] keras
